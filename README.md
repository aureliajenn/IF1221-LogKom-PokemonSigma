# Pokemon Sigma - Tugas Besar IF1221 Logika Komputasional

A text-based Pokémon game developed using **declarative programming** with Prolog. This project demonstrates the use of **logic programming** for modeling game rules, decision-making, and data representation in a fully declarative style.

---

## Features

- Encounter and capture wild Pokémon  
- Turn-based battles with multiple attack moves  
- Damage calculation and type effectiveness  
- Management of player's Pokémon and inventory  

---

## Getting Started
### Prerequisites
GNU-Prolog or any compatible Prolog interpreter installed on your system.  

### Installation
1. Clone the repository:
```bash
git clone https://github.com/aureliajenn/IF1221-LogKom-PokemonSigma.git
```
2. Navigate to the project directory:
```bash
cd IF1221-LogKom-PokemonSigma
```
3. Open the main Prolog file (e.g., main.pl) in your Prolog interpreter.

---

## How to Play

1. Start the game by loading the main Prolog file.
2. Follow the in-game prompts to encounter wild Pokémon, choose attacks, and manage your Pokémon team.
3. Capture Pokémon to add them to your collection and train them for battles.
4. Compete against wild Pokémon to test your strategy and Pokémon knowledge.
